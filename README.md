# CONSULTA-SISTEMA-DE-COMUNICA-O-DE-INFORMA-O-HOSPITALAR-E-AMBULATORIAL-CIHA
CONSULTA SISTEMA DE COMUNICAÇÃO DE INFORMAÇÃO HOSPITALAR E AMBULATORIAL – CIHA
Consulta no site (http://ciha.datasus.gov.br/) para verificar os reports de produção dos hospitais por CNES.
Este procedimento garante que a receita federal terá as informações relevantes para analisar a questão tributária dos hospitais.

Sobre este fluxo de automação:
Consulta CIHA – Sistema de Comunicação de Informação Hospitalar e Ambulatorial

Este workflow permite que clientes da área da Saúde possam fazer uso das consultas ao site http://ciha.datasus.gov.br/ dos seus reports de produção. Este workflow pode ser estruturado para consultar vários CNES (Cadastro Nacional de Estabelecimentos de Saúde) e salvar o resultado em arquivos Excel, Banco de dados, o que foi melhor para que o cliente possa fazer uso destes dados posteriormente.
